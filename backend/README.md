# EmotiCore Multi-Modal Emotion Detection API

This space hosts the backend for EmotiCore, providing robust emotion analysis for Text, Audio, and Images.

## Local Setup
If you want to run this locally:
```bash
cd backend
pip install -r requirements.txt
python main.py
```
